4.20 Emphasis Techniques
1. Use all methods of emphasis sparingly.
2. Apply emphasis techniques consistently across all technical procedures.
a. In this standard a consistent method for emphasis is applied to examples
from technical procedures. This is not meant to imply that this is the only
method that is acceptable, as bounded by other requirements for
emphasis within this standard.
3. Emphasis should be used for the following items, and a different emphasis
should be applied to each:
• Action verbs
• Conditional and logic terms
• Component positions and noun names
• Notes, cautions, and warnings
4. Acceptable emphasis techniques include but are not limited to, capitalization,
bolding, and underlining as shown in Figure 39, Sample Emphasis Techniques.
